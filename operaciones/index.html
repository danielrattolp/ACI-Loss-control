    Notas de metodo

    Trazabilidad

    • Secuencia por tanque: TOV - FW = GOV; GOV x VCF x CTSh = GSV; GSV x (1 - BSW%) = NSV.
    • El VCF automatico replica la formula encontrada en el archivo F.C.V. para Tablas 6A, 6B, 6C y 6D. Mantenga VCF override disponible para valores verificados por software/tablas oficiales.
    • El volumen TOV debe venir de tablas de calibracion/ullage del buque o tanque. Esta hoja no reemplaza las tablas de aforo.
