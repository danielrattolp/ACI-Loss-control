# Paquete Operacional ACI LATAM

## Estado

Version 1.0 para aprobacion e implementacion piloto. Este paquete no declara certificacion ISO 9001 ni debe considerarse vigente hasta completar aprobacion, capacitacion y prueba simulada.

## Documentos

1. `ACI-OP-000-Indice-Operacional.docx`
   - Indice y secuencia de implementacion del sistema operacional.
2. `ACI-MO-001-Manual-Operacional-Loss-Control-Expediting.docx`
   - Metodo completo para recibir, planificar, ejecutar, reportar y cerrar servicios.
3. `ACI-PT-001-Control-Cantidad-Reconciliacion.docx`
   - Medicion, calculos, reconciliacion, diferencias y revision tecnica.
4. `ACI-PT-002-Muestreo-Calidad-Cadena-Custodia.docx`
   - Muestreo, etiquetado, sellado, custodia y revision documental de calidad.
5. `ACI-PT-003-Expediting-Control-Tiempos.docx`
   - Hitos, cronologia, demoras, alertas y control de fuentes.
6. `ACI-PT-004-Evidencia-Reportabilidad.docx`
   - Fotografias, expedientes, integridad, reporte preliminar e informe final.
7. `ACI-PT-005-Diferencias-Soporte-Reclamos.docx`
   - Preservacion, investigacion, hipotesis e informe tecnico de diferencias.
8. `ACI-HSE-001-Seguridad-Emergencias.docx`
   - Seguridad en terreno, autoridad para detener y respuesta a emergencias.
9. `ACI-PC-001-Capacitacion-Habilitacion.docx`
   - Malla formativa, evaluacion, practica supervisada y habilitacion por competencia.
10. `ACI-PQ-001-Paquete-Formatos-Operacionales.xlsx`
   - ACI-F-001 Orden de servicio.
   - ACI-F-002 Checklist operacional.
   - ACI-F-003 Bitacora de eventos.
   - ACI-F-004 Registro fotografico.
   - ACI-F-005 Reporte preliminar.
   - ACI-F-006 Informe final.
11. `ACI-PQ-002-Control-Documental-Competencia.xlsx`
   - ACI-F-101 Listado maestro de documentos.
   - ACI-F-107 Matriz de retencion documental.
   - ACI-F-201 Perfil de cargo.
   - ACI-F-202 Matriz de competencia y habilitacion.
   - ACI-F-203 Registro de capacitacion.
   - ACI-F-204 Evaluacion de desempeno operacional.

## Implementacion obligatoria

1. Designar responsables nominales de Operaciones, Calidad, revision tecnica y aprobacion.
2. Revisar plazos, criterios de alerta, retencion y requisitos legales por pais.
3. Aprobar el manual y los formatos mediante el listado maestro.
4. Capacitar a coordinadores, inspectores y revisores.
5. Ejecutar una operacion simulada completa.
6. Registrar correcciones y emitir la version vigente.
7. Utilizar el paquete en operaciones piloto bajo supervision.

## Regla de uso

Crear una copia del libro operacional por cada orden de servicio. Los originales maestros deben mantenerse protegidos y bajo control de version.
